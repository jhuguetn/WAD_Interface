De 2 create*.cmd scripts kunnen worden gebruikt om een schone DCM4CHEE en/of IQC database aan te maken.
De huidige scripts gaan uit van een installatie in c:\WAD-software in combinatie met xampp.
- pas eventueel de paden (mysql/dcm4chee/dcm4chee_arr) aan in beide cmd-files.
- vul eventueel het mysql root-ww in (of vul deze in zodra erom gevraagd wordt)

Voor linux zijn er 2 shell-scripts meegeleverd met dezelfde functionaliteit.
